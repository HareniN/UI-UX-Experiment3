# UI-UX-Experiment3

## Aim:

To perform a heuristic evaluation of two competing shopping applications, Amazon and Flipkart, compare their user experience (UX), identify usability problems, and propose improved interface designs.

## Algorithm:

1.Select two competing applications for comparison.

2.Study the existing UI and UX of both apps.

3.Analyze the apps using heuristic evaluation principles such as:
Navigation,
Consistency,
Accessibility,
Visual clarity,
User control

4.Identify usability problems in the existing designs.

5.Compare the strengths and weaknesses of both apps.

6.Design improved “After Update” interfaces with better usability.

7.Evaluate the updated designs and compare them with the original versions.

8.Document the improvements and final results.

## Output:

**Apps Compared:**

Amazon and Flipkart

Both apps are popular e-commerce platforms used for online shopping, payments, product browsing, and order tracking.

**Amazon App:**

**Before Update**

1.Homepage was crowded with many banners and offers

2.Navigation looked complex

3.Customer support was difficult to find

4.Too much information caused visual clutter

**After Update**

1.Cleaner and simpler homepage

2.Better spacing and organized categories

3.Easier navigation

4.Customer support made more visible

The updated Amazon design improves readability, reduces clutter, and provides a smoother shopping experience.
